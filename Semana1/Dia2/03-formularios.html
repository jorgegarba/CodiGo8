Elementos de los Formularios

Lista de Elementos

Nro Nombre del Elemento Descripción
1 input Es un campo de entrada de muchos tipos de datos
2 textarea Es un campo de entrada de tipo texto, puede tener más de una fila
3 select
  • Es un elemento con una lista de opciones para elegir
  • Cada opción, es un elemento option
4 button
  • Es un elemento que representa a un botón
5 label
  • Es un elemento que crea una etiqueta asociada a un campo de entrada
6 fieldset
  • Es un elemento que agrupa los campos de un formulario
7 legend
  • Representa al título de algunas etiquetas
  • Por lo general, acompaña al fieldset

Tipos de input

Lista de tipos de input

Número Tipo Descripción
1 text
  • Input para ingresar texto plano
  • Por ejemplo, nombres, apellidos, direcciones, etc.
2 email
  • Input para ingresar correos electrónicos
  • A pesar de ser texto plano, se coloca tipo email para que el formulario valide el ingreso de un correcto email, es decir, con símbolo de @ y un .
3 number
  • Input para ingresar números enteros y/o decimales
4 date
  • Input para ingresar fechas
5 password
  • Input para ingresar contraseñas
  • El navegador, automáticamente, cambia el texto ingresado, por simbolos.
6 time
  • Input para ingresar horas y minutos
7 color
  • Input para seleccionar un color de una paleta de colores
8 checkbox
  • Input para selccionar o desseleccionar elementos de mas de un grupo de checkboxes
  • Es posible seleccionar mas de un checkbox a la vez
9 radio
  • Input para seleccionar un elemento de un grupo de radio buttons
  • Solo se puede seleccionar uno a la vez
10 file
  • Input para seleccionar un archivo local y mandarlo al servidor
11 reset
  • Input en forma de un botón.
  • Sirve para limpiar todos los campos del formulario en el que se encuentre
12 submit
  • Input en forma de un boton
  • Sirve para enviar TODOS los datos del formulario al servidor
